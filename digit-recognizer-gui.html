GUI Digit Recognizer

Machine Learning, Keras, GUI Programming

The goal of this project was to create a machine learning model that would identify a number given to it from 0 to 9. This model is trained using the keras MNIST dataset of 60,000 grayscale images of the ten digits along with a test set of 10,000. Below you can see the model in action.
